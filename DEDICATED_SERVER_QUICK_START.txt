================================================================================
                MOVING TO DEDICATED SERVER - QUICK START
================================================================================

                    Follow these 3 simple phases:

================================================================================


PHASE 1: GET A SERVER (1-5 days)
---------------------------------

   [ ] Send email to IT requesting a VM
       ‚Üí Use template in: IT_REQUEST_TEMPLATE.txt
       ‚Üí Copy and paste into email
       ‚Üí Send to your IT help desk
   
   [ ] Wait for IT to provision the server
       ‚Üí They'll give you: IP address, server name, login
   
   [ ] Connect via Remote Desktop
       ‚Üí Press Windows key + R
       ‚Üí Type: mstsc
       ‚Üí Enter server name/IP
       ‚Üí Login with credentials


================================================================================


PHASE 2: SET UP THE SERVER (1-2 hours)
---------------------------------------

   [ ] Install Python 3.11+
       ‚Üí Download from python.org
       ‚Üí Check "Add to PATH"
       ‚Üí Install for all users
   
   [ ] Copy chatbot files to server
       ‚Üí Copy walmart-hr-chatbot folder
       ‚Üí Put in: C:\Apps\walmart-hr-chatbot
   
   [ ] Install dependencies
       ‚Üí Open Command Prompt on server
       ‚Üí Run:
         cd C:\Apps\walmart-hr-chatbot\backend
         python -m venv .venv
         .venv\Scripts\activate
         pip install fastapi uvicorn openai python-dotenv pydantic httpx python-docx
   
   [ ] Test it works
       ‚Üí Run: uvicorn main:app --host 0.0.0.0 --port 8000
       ‚Üí Open browser: http://localhost:8000/health
       ‚Üí Should see: {"status":"healthy"}
       ‚Üí Press Ctrl+C to stop


================================================================================


PHASE 3: CONFIGURE AUTO-START (30 minutes)
-------------------------------------------

   [ ] Set up Windows Task Scheduler
       ‚Üí Open Task Scheduler
       ‚Üí Create Task (not Basic Task)
       ‚Üí Name: LAX2 HR Chatbot Service
       ‚Üí Run whether user is logged on or not
       ‚Üí Run with highest privileges
       ‚Üí Trigger: At startup
       ‚Üí Action: Start program
       ‚Üí Program: C:\Apps\walmart-hr-chatbot\START_CHATBOT_SERVICE.bat
   
   [ ] Configure Windows Firewall
       ‚Üí Open PowerShell as admin
       ‚Üí Run:
         New-NetFirewallRule -DisplayName "LAX2 HR Chatbot" -Direction Inbound -LocalPort 8000 -Protocol TCP -Action Allow
   
   [ ] Get server IP address
       ‚Üí Run: ipconfig
       ‚Üí Write down IPv4 Address: ___________________
   
   [ ] Update chatbot.html with server IP
       ‚Üí Run: UPDATE_CHATBOT_IP.bat
       ‚Üí Enter the server IP
       ‚Üí File is updated automatically!
   
   [ ] Test from another computer
       ‚Üí Open updated chatbot.html
       ‚Üí Ask a question
       ‚Üí Should work!
   
   [ ] Send update to your team
       ‚Üí Attach updated chatbot.html
       ‚Üí Tell them it's now 24/7!
